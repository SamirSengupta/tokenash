Real-time aggregate metrics from Tokenash proxy instances worldwide. All data is anonymous — only token counts, compression ratios, and cost estimates are collected. Telemetry is off by default; [opt in](https://github.com/SamirSengupta/tokenash/blob/main/tokenash/telemetry/beacon.py) with `TOKENASH_TELEMETRY=on`.
